# Flask CI/CD Project

I developed a Flask web application using Python and implemented a complete CI/CD workflow.

## Tech Stack Used
- Python
- Flask
- Docker
- GitHub Actions (CI/CD)

## Project Overview

This project demonstrates a full Continuous Integration and Continuous Deployment (CI/CD) pipeline.

- The Flask application is containerized using Docker.
- GitHub Actions is used to automate the CI/CD process.
- Whenever code is pushed to the GitHub repository:
  - The CI pipeline is triggered automatically.
  - The application is built and tested.
  - Docker image is created.
  - Deployment process is executed.
  - Test results and deployment status are generated.

This ensures automated testing, consistent builds, and reliable deployments.
